## Overview

The Flush feature is used to manually empty InnoPick of certain products.

- This can be useful when trying to free up space in InnoPick, or to empty out expired products, or limited-time promotional products that are no longer valid to be sold.
- **Note**: This feature creates a special Truckload so it is not recommended to use it in the middle of production.

## Confirming a Flush

When a product is selected to be Flushed, a confirmation pop-up will appear.

<img src="../images/inventory/flush-confirmation.png" width="750" alt="Flush Confirmation Pop-up" />

<hr style={{ border: "none", height: "2px", backgroundColor: "#a8a8a8", margin: "2em 0" }} />

## Flush Truckload

If the Flush operation is confirmed, a custom Truckload is created and launched; and InnoPick receives the case sequence from MixMaster.

- The Flush Truckload will be visible in the [Truckloads](../truckloads/index.md) page:

<img src="../images/inventory/flush-truckload.png" width="1400" alt="Flush Truckload in Truckloads Page" />

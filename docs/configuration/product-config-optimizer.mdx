## Overview

The Product Configuration Optimizer is a feature that intelligently analyzes the past or upcoming production statistics to make recommendations on which products should be slotted in InnoPick.
- InnoPick is configured to handle only a certain number of products. Exceeding this limit increases the chance of deadlocks and other problems.
- The Product Configuration Optimizer analyzes which products represent the most production volume, and whether they are assigned to InnoPick.

## Recommendations

- After having analyzed the current configuration in light of recent production statistics, it makes recommendations.
    - Check the **Show recommendations only** checkbox to see what changes the optimizer is suggesting.
    - In the screenshot below, the Optimizer has identified 4 products which have had zero production volume within the last 7 days. It recommends removing these from InnoPick.

## Optimizer Mode

The **Mode** drop-down menu is where the user can select which production statistics to optimize the system on.

<img src="../images/configuration/optimizer-mode.png" width="425" alt="Optimizer Mode Selection" />

<hr style={{ border: "none", height: "1px", backgroundColor: "#747474", margin: "2em 0", width: "600px" }} />


**Note**: The Optimizer results can help users and production supervisors to improve the efficiency of InnoPick, especially during periods when products are being introduced or removed, or when the volume profile changes significantly.

**Tip**: Use the **Pending orders from current delivery date** Mode before the start of production to see if making some changes to the InnoPick-assigned products could help produce the day's orders more efficiently.

## Creating a New User

When creating a new user account, the following information is required.

<img src="../images/configuration/create-user.png" width="950" alt="Create New User Form" />

Each user is given a Role, which can be one of the following options: Admin, Manager, Clerk, or Terminal User. Each has different levels of access:

-   **Admin**: Full access to all controls and options.

-   **Manager**: Access to all operational settings, but not site settings.

-   **Clerk**: Access necessary to operate the system for production, but not change settings.

-   **Terminal User**: Restricted login to Manual Induction station or Depal screens.

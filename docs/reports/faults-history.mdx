# Faults History

**[Home](../index.md) > [Reports](index.md) > Faults History**

## Overview

This report lists all the faults that have been reported to MixMaster.

- MixMaster has its own faults that it generates, but it also receives fault information from InnoPick and possibly other sub-systems.
- The faults can be filtered and searched.
